# 项目简介

## 核心功能
- 用户首页
- 心理咨询师展示
- 心理文章阅读
- 交流论坛互动
- 心理测试体验
- 心理资讯浏览
- 个人中心管理

## 适用场景
- 高校心理健康教育
- 大学生心理健康维护
- 心理咨询资源整合

## 技术特点
- 架构：B/S、MVC、Web框架
- 开发环境：IDEA、JDK1.8、Mysql、Maven、Navcat
- 技术栈：Java、SpringBoot、Vue
